# Minds & Machines

AI alignment, moral agency, superintelligence, and the futures we might build.

Part of the [Minds & Machines](https://metafunctor.com/series/minds-and-machines/) series on [metafunctor.com](https://metafunctor.com).

## Overview

What happens when we build minds we can't understand?

This series collects essays on AI alignment, moral agency, superintelligence, and the philosophical foundations needed to think clearly about minds -- both artificial and human. It spans speculative fiction, technical essays, and philosophy.

## Building

```bash
make docs        # Build the site
make docs-serve  # Preview at localhost:8000
```

Requires Python with `mkdocs` and `mkdocs-material`.
